How /lfg works

  • /lfg — post a squad (mission, role, MR, voice optional).
  • /lfg_list — browse open posts in the server.
  • Members react or reply to join; host marks filled when complete.

Staff can route posts to a dedicated channel via setup. See commands for the full list.

Tips for hosts

  • State mission, goal (e.g. “steel path”, “quick clear”), and if voice is required.
  • Close the post when full — reduces ping fatigue.
  • Link world state for fissure tier or sortie element.
