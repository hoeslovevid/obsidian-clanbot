Events

Two kinds: Digital Extremes world events in-game, and your clan’s schedule on Discord.

Warframe world events

Plague Star, Thermia, Nightwave acts, and similar rotations show on our world state board. Wire alert channels with /wfnotify configure so your clan does not miss them.

Discord /events

  • /events — create a server event with time, description, and RSVP buttons.
  • /events_list — what’s coming up this week.
  • Reminders ping interested members before start — great for sortie nights and dojo sessions.

Giveaways

Clan giveaways (plat, slots, fashion prizes) run through the bot and dashboard when a mod is logged in. Create entries, set end time, and roll winners without a third-party bot.
