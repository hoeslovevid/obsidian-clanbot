FAQ

Quick answers about invites, permissions, Warframe data, and the dashboard.

Is Obsidian Overseer free?

Yes. Invite the bot and use the public feature set. Optional paid hosting is not required for normal clan use.

What permissions do I need to add the bot?

You need Manage Server on the Discord server. The invite link requests the bot scopes and permissions needed for slash commands, moderation helpers, and voice features.

Slash commands are missing — what now?

Wait a minute after invite, then try again. Staff can run a command sync if available. Re-inviting with the same link often refreshes command registration. Also confirm the bot role can see the channel you are typing in.

Where does Warframe data come from?

World-state features (Baro, cycles, sortie, fissures, and related feeds) use public Warframe world-state APIs. If a panel shows degraded status, the upstream API may be slow or unavailable — try again shortly.

How do I turn modules on or off?

Many modules can be toggled per server from staff tools or the mod dashboard Features tab. New servers start lean so music, pets, and gambling stay off until you opt in.

Why does the dashboard open on Railway?

The dashboard talks to the live bot API. Hosting it on the same origin as the API avoids browser CORS issues. Bookmark obsidianoverseer.up.railway.app/dashboard.html or use Dashboard in the site nav.

Who can use the mod dashboard?

Sign in with Discord. You will see servers where you have Manage Server (and the bot is present). Members without that permission will not manage guild settings.

Is this affiliated with Digital Extremes or Discord?

No. Obsidian Overseer is a fan-made community bot and is not affiliated with Discord or Digital Extremes / Warframe.

How do I get support?

Join our Discord, open a ticket with /ticket on a configured server, or use the contact form. Include your server name and steps to reproduce for bugs.
