FAQ

Quick answers about invites, permissions, Warframe data, and the dashboard.

Is Obsidian Overseer free?

Yes. Invite the bot and use the public feature set.

What permissions do I need to add the bot?

You need Manage Server. The invite requests scopes for slash commands, moderation helpers, and voice features.

Slash commands are missing — what now?

Wait a minute after invite. Confirm the bot can see the channel. Staff can sync commands, or re-invite with the same link.

Where does Warframe data come from?

World-state features use public Warframe APIs. If a panel looks degraded, try again shortly — upstream may be slow.

How do I turn modules on or off?

Many modules toggle per server from staff tools or the mod dashboard Features tab. New servers start lean.

Why does the dashboard open on Railway?

It talks to the live bot API on the same origin to avoid browser CORS issues. You can later point a custom domain (e.g. dash.obsidianoverseer.com) at that host and set DASHBOARD_PUBLIC_URL in site config.

Who can use the mod dashboard?

Sign in with Discord. You see servers where you have Manage Server and the bot is present.

Is this affiliated with Digital Extremes or Discord?

No. Obsidian Overseer is a fan-made community bot.

How do I get support?

Join our Discord, open a ticket with /ticket on a configured server, or use the contact form.
