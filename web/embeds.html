Embed gallery

Drop these into OBS Browser Sources or clan dashboards. Add ?embed=1 to strip site chrome; &tv=1 bumps type for living-room displays.

Embed vs TV mode

  • embed=1 — minimal chrome, no nav. Best for iframes and OBS.
  • tv=1 — larger text and spacing for Baro/trader panels on a TV.
  • Use full URLs below; swap the host if you self-host the site.
