Setup guide

Get Obsidian Overseer running on your clan Discord in a few minutes.

Add to Discord Open dashboard

1. Invite the bot

Use Add to Discord with an account that has Manage Server. Accept the permissions prompt so slash commands and moderation tools can work.

  • Commands usually appear within a minute after invite.
  • If they are missing, try a staff sync or re-invite with the same link.

2. First commands

  1. Run /menu — hub for daily actions and help.
  2. Run /help — search features and commands.
  3. Run /baro — confirm Warframe world-state is healthy.

3. Wire Warframe notifications

Pick channels for alerts and feeds, then configure notify:

  • /wfnotify configure — alerts, cycles, and related feeds
  • Optional: forum / YouTube / devstream channels as your clan prefers

4. Tickets, voice, and community

  • Set a ticket category / staff role so /ticket routes to mods.
  • Enable join-to-create voice if your clan uses temp VCs.
  • Turn on LFG, suggestions, or events when ready — many modules start lean.

5. Mod dashboard

Staff with Manage Server can open the web dashboard for inbox, Warframe status, setup health, features, and giveaways.

Sign in with Discord OAuth. Prefer the Railway-hosted URL (or a custom domain you point at it) so the API stays same-origin.
